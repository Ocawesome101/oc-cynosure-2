CEX: Cynosure Executable format
==============================

Numbers are little-endian.

4 bytes: magic number 0x43796e6f
1 byte: flags:
  0x1   | lua 5.3 only
  0x2   | static linked (libraries bundled)?
  0x4   | is bootable image
  0x8   | is executable
  0x10  | is library
  0x20  | big-endian (otherwise little-endian)
  0x40  | unused
  0x80  | unused
1 byte: OS ID:
  0   | Cynosure 2.0
  1   | OpenOS
  2   | Tsuki
  3   | Fuchas
  4   | MiniOS NT
  5   | PsychOS
  6   | Linux
  7   | macOS
  8   | Windows
  255 | N/A
1 byte: number of links
for each link:
  1 byte: length of library name
  N bytes: library name
2 bytes: length of data
/*
 * In the case of static linking, linked-in
 * libraries should be part of the data section.
 */
N bytes: data
